# p4-ts

Typed TypeScript helpers for working with the Perforce `p4` CLI.

## Packages

| Package | Description | Status |
|---------|-------------|--------|
| [`packages/core`](./packages/core) | Runtime library for running and parsing `p4` commands | Available |

## Development

```bash
bun install
bun run typecheck

bun run build
```
